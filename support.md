  ## Common questions

  **How do I add a second child?**
  Premium subscription required. Open the Parent profile → Children section → Add a Child.

  **My kid can't sign in.**
  Make sure your child entered the correct 6-digit code shown in the parent profile. Codes are case-sensitive.

  **How do I delete my account?**
  Parent profile → Delete Account. This cascades all family data and is irreversible.

  **My subscription isn't working.**
  In the Parent profile → Tap "Restore Purchases". If still stuck, email us.

  ## Report a bug
